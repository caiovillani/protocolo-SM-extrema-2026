## 6. METODOLOGIA DE CONVERSÃO

### 6.1. Ferramenta Utilizada
- **Biblioteca:** python-docx (Python)
- **Script:** `gerar_documentos_oficiais.py`
- **Linguagem:** Python 3.x

### 6.2. Processo Executado
1. Leitura do arquivo fonte `RASCUNHO_INSTRUMENTOS_JURIDICOS.md`
2. Divisão em duas partes:
   - PARTE I: Termo de Referência (linhas 18-567)
   - PARTE II: Plano de Trabalho (linhas 569-1279)
3. Parsing de Markdown com reconhecimento de:
   - Cabeçalhos hierárquicos (# ## ### ####)
   - Tabelas (| coluna | coluna |)
   - Listas (numeradas e com marcadores)
   - Texto em negrito (**texto**)
   - Linhas horizontais (---)
4. Aplicação de formatação profissional
5. Geração de arquivos .docx com padrões governamentais
