- **pips_models.py** — Dataclasses for PIPS state management

**Testing:** 85 test cases for PIPS reliability (tests/test_pips.py)

### J2. PIPS — Protocolo de Processamento Iterativo com Persistência de Estado

**Purpose:** Handle long-duration tasks (3+ files, >50,000 tokens) with state persistence

**Architecture:**
```
.pips/projeto_<nome>/
├── _config/    # Objetivo imutável (Estrela do Norte)
├── _state/     # Estado de processamento (progress.yaml)
├── _output/    # Insights e entregas (insights_raw.md)
└── _source/    # Arquivos fonte
```

**Lifecycle:** Work → Save → Validate → Reset → Resume

**Infinite Memory Protocol Integration:**
- PreCompact hook: Snapshot active projects before context compaction
- SessionStart hook: Display resumable projects at session start
- Stop hook: Final snapshot on session end

**Recovery Strategy:** Backup → Checkpoints → Minimum viable state (graceful degradation)
