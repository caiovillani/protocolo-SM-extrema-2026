# POP-05: ELABORAÇÃO DO PROJETO TERAPÊUTICO SINGULAR (PTS)

**Secretaria Municipal de Saúde de Extrema/MG**
**Versão:** 1.0 | **Data:** Janeiro/2026

---

## 1. OBJETIVO
Padronizar a elaboração do Projeto Terapêutico Singular para usuários com demandas de saúde mental.

## 2. DEFINIÇÃO
O PTS é um instrumento de planejamento do cuidado, construído de forma compartilhada entre equipe, usuário e família, que organiza as intervenções singularizadas para cada caso.

## 3. QUANDO ELABORAR

| Situação | Obrigatoriedade |
|----------|-----------------|
| Transtorno mental grave e persistente | Obrigatório |
| Casos com classificação AMARELO ou superior | Obrigatório |
| Casos em compartilhamento APS-AES | Obrigatório |
| Usuários do CAPS (qualquer modalidade) | Obrigatório |
| Depressão/ansiedade leve em tratamento | Recomendado (simplificado) |

## 4. RESPONSÁVEIS
- **Elaboração:** Profissional de referência do caso (médico, enfermeiro, psicólogo)
- **Participação:** Equipe multiprofissional, usuário, família/cuidador
- **Validação:** Coordenador técnico do serviço

## 5. COMPONENTES DO PTS

### 5.1 Diagnóstico Situacional
- Diagnóstico(s) clínico(s) - CID-10
- Classificação de risco atual
- Contexto familiar e social
- Rede de apoio identificada
- Vulnerabilidades e potencialidades
- Determinantes sociais relevantes

### 5.2 Objetivos Terapêuticos
- **Curto prazo (30 dias):** Metas imediatas, estabilização
- **Médio prazo (90 dias):** Metas de tratamento, funcionalidade
- **Longo prazo (>90 dias):** Metas de reabilitação, autonomia

**Características dos bons objetivos (SMART):**
- Específicos
- Mensuráveis
- Alcançáveis
- Relevantes
- Temporais

### 5.3 Intervenções Pactuadas

| Tipo | Exemplos |
|------|----------|
| **Farmacológicas** | Medicações, doses, posologia, tempo |
| **Psicoterapêuticas** | Psicoterapia individual, TCC, grupos |
| **Psicossociais** | Oficinas, atividades, convivência |
| **Clínicas** | Acompanhamento de comorbidades |
| **Sociais** | Articulação CRAS, benefícios, moradia |
| **Educacionais** | Apoio escolar, PSE |
| **Ocupacionais** | Trabalho protegido, geração de renda |

### 5.4 Responsabilidades Definidas
- O que CADA PROFISSIONAL faz
- O que o USUÁRIO se compromete
- O que a FAMÍLIA/REDE faz
- Profissional de REFERÊNCIA do caso

### 5.5 Cronograma de Reavaliação
- Datas de retorno
- Momentos de reavaliação do PTS
- Critérios para revisão

## 6. PROCEDIMENTO PASSO A PASSO

### ETAPA 1: PREPARAÇÃO
| Passo | Ação |
|-------|------|
| 1.1 | Reunir informações do prontuário |
| 1.2 | Identificar profissionais envolvidos no caso |
| 1.3 | Agendar reunião de discussão (equipe + usuário) |
| 1.4 | Preparar formulário de PTS |

### ETAPA 2: DISCUSSÃO EM EQUIPE (30-45 min)
| Passo | Ação |
|-------|------|
| 2.1 | Apresentar o caso (profissional de referência) |
| 2.2 | Cada profissional contribui com sua visão |
| 2.3 | Identificar necessidades prioritárias |
| 2.4 | Propor intervenções |
| 2.5 | Definir responsabilidades |
| 2.6 | Elaborar rascunho do PTS |

### ETAPA 3: PACTUAÇÃO COM USUÁRIO E FAMÍLIA
| Passo | Ação |
|-------|------|
| 3.1 | Apresentar proposta de PTS ao usuário |
| 3.2 | Escutar sugestões e preocupações |
| 3.3 | Ajustar conforme pactuação |
| 3.4 | Definir metas em conjunto |
| 3.5 | Registrar compromissos de cada parte |
| 3.6 | Entregar cópia ao usuário |

### ETAPA 4: REGISTRO E MONITORAMENTO
| Passo | Ação |
|-------|------|
| 4.1 | Registrar PTS em prontuário |
| 4.2 | Compartilhar com equipe (e-mail, sistema) |
| 4.3 | Se compartilhado: enviar para outro serviço |
| 4.4 | Monitorar execução das intervenções |
| 4.5 | Reavaliar nas datas previstas |

## 8. REVISÃO DO PTS

### Quando revisar
- Conforme cronograma estabelecido
- Mudança significativa do quadro clínico
- Alcance ou não dos objetivos
- Solicitação do usuário/família
- Contrarreferência de outro serviço

### Como revisar
1. Avaliar cumprimento das metas
2. Identificar facilitadores e barreiras
3. Ajustar intervenções conforme evolução
4. Redefinir objetivos se necessário
5. Registrar nova versão
